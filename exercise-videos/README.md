# Muscle Exercise Videos

Add provided muscle exercise tutorial videos in this folder as `.mp4` files.

The app expects file names based on the exercise name in lowercase with spaces converted to hyphens.

Examples:

- `push-up.mp4`
- `bench-press.mp4`
- `lat-pulldown.mp4`
- `barbell-row.mp4`
- `squat.mp4`
- `romanian-deadlift.mp4`
- `plank.mp4`

When a client clicks a Muscle Exercises card, The Fit Geek loads the matching local video file only. It does not open YouTube or any external search.
