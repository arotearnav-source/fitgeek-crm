# Yoga Videos

Add provided yoga tutorial videos in this folder as `.mp4` files.

The app expects file names based on the Sanskrit asana name in lowercase with spaces converted to hyphens.

Examples:

- `tadasana.mp4`
- `urdhva-hastasana.mp4`
- `adho-mukha-svanasana.mp4`
- `virabhadrasana-i.mp4`
- `shavasana.mp4`

When a client clicks an asana card, The Fit Geek loads the matching local video file only. It does not open YouTube search.
